'고리가 n그램일 때 몇 개의 고리를 풀어야 할까?'와 'n개의 고리를 풀면 최대 몇 그램까지 표현할 수 있을까?' 두 가지 시작점을 생각할 수 있는데 나는 첫 번째 관점으로 접근하다가 8그램을 손수 그려볼 때쯤 두 번째 관점을 떠올렸다. 두 번째 관점으로 접근하여 고리 갯수를 1부터 점점 늘려서 계산해보면 규칙을 찾을 수 있다.

# 🔥 해결방법
**고리 1개를 풀 때마다 1그램짜리 고리 1개가 생겨나는 것**에 주의하며 규칙성을 따져본다. 쉽게 표현하기 위해 ⬛️을 풀린 1그램짜리 고리, ⬜️은 연속된 갯수만큼 연결된 고리라고 가정한다.

## 고리를 1개 풀었을 때
- 1그램짜리 고리 1개
- 총 3부분(❓⬛️❓)

1 = **1그램**   
2 = **2그램**  
3 = 2그램 + 1그램  
4 = **4그램**  
5 = 4그램 + 1그램  
6 = 4그램 + 2그램  
7 = 4그램 + 2그램 + 1그램  

표현할 수 있는 최대 숫자는 7이며, ⬜️⬜️⬛️⬜️⬜️⬜️⬜️로 표현할 수 있다.

## 고리를 2개 풀었을 때
- 1그램짜리 고리 2개
- 총 5부분(❓⬛️❓⬛️❓)

1 = **1그램**  
2 = 1그램 + **1그램**  
3 = **3그램**  
4 = 3그램 + 1그램  
5 = 3그램 + 1그램 + 1그램  
6 = **6그램**  
7 = 6그램 + 1그램  
8 = 6그램 + 1그램 + 1그램  
9 = 6그램 + 3그램  
10 = 6그램 + 3그램 + 1그램  
11 = 6그램 + 3그램 + 1그램 + 1그램  
12 = **12그램**  
13 = 12그램 + 1그램  
14 = 12그램 + 1그램 + 1그램  
15 = 12그램 + 3그램  
16 = 12그램 + 3그램 + 1그램  
17 = 12그램 + 3그램 + 1그램 + 1그램  
18 = 12그램 + 6그램  
19 = 12그램 + 6그램 + 1그램  
20 = 12그램 + 6그램 + 1그램 + 1그램  
21 = 12그램 + 6그램 + 3그램  
22 = 12그램 + 6그램 + 3그램 + 1그램  
23 = 12그램 + 6그램 + 3그램 + 1그램 + 1그램  

표현할 수 있는 최대 숫자는 23이며, ⬜️⬜️⬜️⬛️⬜️⬜️⬜️⬜️⬜️⬜️⬛️⬜️⬜️⬜️⬜️⬜️⬜️⬜️⬜️⬜️⬜️⬜️⬜️로 표현할 수 있다.

## 고리를 n개 풀었을 때
- 고리 1그램짜리 n개
- 총 (2n-1)부분(❓⬛️❓⬛❓...❓⬛❓)

1그램짜리 고리가 **n**개이므로 n까지는 1그램짜리들로 표현할 수 있다. 따라서, 그 다음에 필요한 숫자는 **(n+1)**이 된다. 마찬가지로 (n+1)+n까지 표현이 가능하고 다음에 필요한 숫자는 (n+1)+n+1 = **2(n+1)**이 된다. 이 또한 2(n+1)+(n+1)+n까지 표현이 가능하고 다음에 필요한 숫자는 2(n+1)+(n+1)+n+1 = **4(n+1)**이 된다. 다음은 4(n+1)+2(n+1)+(n+1)+n+1 = **8(n+1)**, 그 다음은 8(n+1)+4(n+1)+2(n+1)+(n+1)+n+1 = **16(n+1)**가 된다. 각 단계에서 표현할 수 있는 최대 숫자는 가지고 있는 모든 요소를 더한 값이 된다. 따라서 고리가 n개일 때 표현할 수 있는 최대 숫자는 식으로 표현하면 `2^(n+1) * (n+1) - 1`가 된다.

# 🔥 주의
- input 범위 항상 확인하자! : 이 문제에서 입력은 전체 고리의 무게이다. 범위는 `3 <= n <= 10^18`인데 아무 생각 없이 n을 int로 선언했다가 한참을 돌아왔다.
